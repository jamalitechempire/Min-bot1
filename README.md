## 🚀 **ABOUT JAMALI TECH MD**

**JAMALI TECH MD** is a premium WhatsApp bot with 350+ commands to automate tasks, moderate groups, download media, and provide AI-powered responses.

### ✨ **FEATURES**

| Category | Features |
|----------|----------|
| **👥 Group Management** | Add, remove, promote, demote members, tagall, tagadmin, close/open group |
| **🔗 Auto Moderation** | Anti-link, anti-spam, anti-bot, anti-badword, anti-sticker, anti-image |
| **🛡️ Anti-Bug & Security** | Detect and block users sending harmful messages, warn system |
| **📥 Media Downloader** | Download from YouTube (MP3/MP4), TikTok, Facebook, Instagram, Twitter |
| **🤖 AI Features** | GPT, Gemini, Blackbox AI, DALL-E image generation, Code generator |
| **🎮 Fun & Games** | Truth or Dare, Quotes, Jokes, Anime reactions, Couple calculator |
| **👋 Welcome/Goodbye** | Custom welcome and goodbye messages for groups |
| **📊 Bot Dashboard** | Real-time admin panel, session management, MongoDB storage |
| **🔄 Auto Features** | Auto view status, auto like status, auto recording, auto react |
| **🔧 Utilities** | Sticker maker, QR code generator, Weather info, YouTube search |
| **👑 Owner Controls** | Block/unblock, broadcast, join/leave groups, restart/shutdown |
